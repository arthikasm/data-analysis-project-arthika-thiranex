## Real-World Data Project: Retail Sales Analysis

## Objective
Analyze retail sales data to identify trends, top-performing products, and sales patterns.

## Domain
Retail

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
1. Dataset exploration
2. Category-wise sales analysis
3. Top-selling products analysis
4. Monthly sales trend analysis

## Key Findings
- Certain product categories contribute more revenue.
- A small number of products generate a large share of sales.
- Sales vary across different months.

## Conclusion
The analysis helps identify profitable categories and products, enabling better business decisions.
